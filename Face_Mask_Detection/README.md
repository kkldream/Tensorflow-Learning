# 口罩偵測-VGG19、MobileNetV2預訓練比較

先使用[MediaPipe](https://google.github.io/mediapipe/)的[Face Detection](https://google.github.io/mediapipe/solutions/face_detection)做人臉辨識，取得人臉ROI區域並做選轉校正，再將此ROI丟到模型做口罩偵測。

## 模型結構

模型結構：


* ACC = 91.6244%
* 混淆矩陣
