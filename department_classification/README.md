# 多元分類實作-星座分類

輸入一日期將返回此日期為何星座，星座標籤共12個，如下：  
* 摩羯座 (12/22~01/19)
* 水瓶座 (01/20~02/19)
* 雙魚座 (02/20~03/20)
* 牡羊座 (03/21~04/20)
* 金牛座 (04/21~05/20)
* 雙子座 (05/21~06/21)
* 巨蟹座 (06/22~07/22)
* 獅子座 (07/23~08/22)
* 處女座 (08/23~09/22)
* 天秤座 (09/23~10/22)
* 天蠍座 (10/23~11/21)
* 射手座 (11/22~12/21)
